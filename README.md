# Phishing Analysis — Reportes de Seguridad

Repositorio de análisis formal de incidentes de phishing.
Formato SOC alineado a ISO/IEC 27035:2023.

## Contenido

- `reporte_phishing.py` — Generador de reportes en PDF
- `Reporte_Phishing_INC-2026-001.pdf` — Primer caso analizado

## ¿Qué incluye cada reporte?

- Datos del incidente con identificador único
- Hallazgos técnicos: SPF, DKIM, DMARC, IP origen
- Técnicas de ataque mapeadas a MITRE ATT&CK
- Indicadores de Compromiso (IOCs)
- Conclusión y recomendaciones

## Referencia normativa

ISO/IEC 27035:2023 — Gestión de Incidentes de Seguridad
